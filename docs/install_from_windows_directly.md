# 🔧 Build from Windows

### 🐍 Python

Install Python >=3.12 from the Microsoft Store.

---

## 📦 Update pip

```powershell
pip install --upgrade pip
```

---

## 🧱 Install ESPHome

```powershell
pip install esphome==2026.7.0
```

---

## 🛠️ Build

To build the firmware in French:
```powershell
build-fr.bat
```

Or in English:
```powershell
build-en.bat
```
